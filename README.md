# The Launch Pad (TLP)

A modular, full-stack space platform supporting real-time launch data, visualizations, and a high-traffic audience.

## Structure
- web/: Frontend (Vite+React+Tailwind)
- api/: Backend (Node.js+Express+Postgres)
- admin/: Admin dashboard (React Admin)
- mobile/: Mobile app (React Native + Expo)
- shared/: Shared code and types
- scripts/: Scripts and utilities

## Getting Started
Each directory contains set-up instructions specific for its module.
